# iceConfirm
带有入场动画和特效的确认框  
点击小方块设置是否跳过动画  
不适用于频繁需要确认的场景  
Confirmation box with entrance animation and special effects  
Click the small box to set whether to skip the animation  
This method is not applicable to scenarios that require frequent confirmation
